# Vortex in a Glass Sphere

A step-by-step process for building a procedural vortex enclosed in a glass sphere using Three.js Shader Language (TSL). Covers the creation of a 2D shader, texture effects, transition to particles, and the addition of a glass material with realistic lighting effects.
